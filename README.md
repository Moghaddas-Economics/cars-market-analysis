# cars-market-analysis
Car market analysis using AutoScout24 dataset - Machine Learning 

##  How to Run This Code (Executable Instructions)

### Prerequisites
- **Kaggle account** (free) - [Sign up here](https://www.kaggle.com)
- No local installation required

### Step-by-Step Execution Guide

#### Step 1: Access the Dataset
The dataset is already uploaded to Kaggle:
- **Dataset URL:** https://www.kaggle.com/datasets/maryammoghaddasbayat/carsmarketanalysis
- **File:** `autoscout24_16and10200.csv` (1.39 MB, 16 columns, 10,200 rows)

#### Step 2: Create a New Kaggle Notebook
1. Go to [kaggle.com](https://www.kaggle.com)
2. Click **"Create"** → **"New Notebook"**
3. Select **"Notebook"** option

#### Step 3: Add the Dataset to Your Notebook
1. On the right side, click the **"Add Data"** button
2. Search for **"carsmarketanalysis"**
3. Click **"Add"** to attach the dataset
4. You will see the dataset appears in `/kaggle/input/carsmarketanalysis/`

#### Step 4: Import the Notebook Code
**Option A: Copy from this repository**
1. Open any RQ notebook file from this repository
2. Copy all the code
3. Paste into your Kaggle notebook

**Option B: Upload directly**
1. In Kaggle notebook, click **File** → **Import Notebook**
2. Upload the `.ipynb` file from this repository

#### Step 5: Run the Notebook
1. Click **"Run All"** from the Cell menu
2. Wait for execution (1-3 minutes per notebook)
3. Results will be saved to `/kaggle/working/`

#### Step 6: Download Results
1. Click the **"Output"** tab on the right side
2. Click **"Download All"** to get all CSV and PDF files
3. Results match the tables and figures in this repository
